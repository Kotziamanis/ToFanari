ToFanari — Portable Release
========================================

Version: v1.5.0

IMPORTANT: Do NOT copy ToFanari.exe alone.
It requires the _internal/ folder and settings/ to run.

USAGE:
  1. Copy this ENTIRE 'ToFanari' folder to your desired location,
     OR unzip ToFanari_portable.zip.

  2. Run ToFanari.exe from inside the folder.

  3. Optional: Run Create_Desktop_Shortcut.bat to add a desktop
     shortcut (points to the exe in this folder, does not copy it).

PORTABLE: This folder is self-contained. Copy the whole folder or
unzip ToFanari_portable.zip on another PC and run from there.

If the app fails to start on a new PC, install Microsoft Visual C++
Redistributable (latest) from microsoft.com.
