# Tofanari Main Tool (logical app)

**Implementation files live at the repository root** next to this folder (`app.py`, `config.py`, `database.py`, `pdf_ops.py`, …). This directory is a **label only** so it is obvious which files belong to the main publishing app versus **PDF Marker** (`pdf_marker/`) and **Synch-tool** (`synch-tool/`).

- **Run:** `python app.py` from the repo root (with venv and `pip install -r requirements.txt`).
- **Do not** confuse with `pdf_marker/` — that is the separate “place marks on PDF” tool.
